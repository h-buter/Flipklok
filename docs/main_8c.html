void main ( void )

Main function that initializes system components and enters low-power mode.

This function is responsible for setting up various system components, including the GPIO, ADC, clocks, and timekeeping system. It also loads the mechanical time from flash memory (if it is connected to a debugger the flash is cleared and the mechanical time must be set manually). It also initializes the UART (if enabled). Additionally, the function sets the time of the mechanical clock and enters low-power mode with interrupts enabled. The system performs various initialization tasks to ensure the proper operation of the application.

It performs the following steps:

  • Stops the watchdog timer to prevent system reset.
  • Initializes UART if UART_ENABLED is defined.
  • Sets up GPIO, ADC, clocks, and timekeeping.
  • Writes the stored mechanical time from flash to UART.
  • If the value is invalid because the debugger overwritten the flash, enters an infinite loop.
  • Configures timers and enters low-power mode with interrupts.
Note
If UART is enabled, a warning will be displayed that the UART will be initialized.
